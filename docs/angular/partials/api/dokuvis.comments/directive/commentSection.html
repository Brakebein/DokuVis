commentSection

  1. - directive in module dokuvis.comments

Directive to show comments of the specified target, to add new comments, or answer to existing ones.

Dependencies

  • Comment
  • Utilities

Directive Info

  • This directive creates new scope.
  • This directive executes at priority level 0.

Usage

  • as element: 
    <comment-section
    target="string"
    type="string">
  ...
</comment-section>

Arguments

Param Type Details
target String

Id of the target instance (e.g. id of document/task)
type String

Type of the target instance, respectively the type of the comment (e.g. 'task', 'picture', 'source')
