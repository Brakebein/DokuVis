fileDropArea

  1. - directive in module dokuvis.utils

A grey area, where someone can drop a file passed to a FileUploader. On click, a file dialog opens to select a file or multiple files.

Dependencies

  • $timeout
  • $state

Directive Info

  • This directive creates new scope.
  • This directive executes at priority level 0.

Usage

  • as element: 
    <file-drop-area
    uploader="FileUploader"
    label="string"
    [ui-sref="string"]
    [multiple]
    [column]>
  ...
</file-drop-area>
  • as attribute: 
    <ANY
  file-drop-area
  uploader="FileUploader"
  label="string"
  [ui-sref="string"]
  [multiple]
  [column]>
...
</ANY>

Arguments

Param Type Details
fileDropArea Boolean
uploader FileUploader

Instance of FileUploader
label String

Some text displayed next to a plus sign
uiSref
(optional)
 String

On click, it doesn't open the file dialog, but it transitions to given state. After adding file via file drop, it transitions to given state.
