imageViewer

  1. - directive in module dokuvisApp

A viewer to inspect an image in detail. The user is able to zoom and pan. It builds on three.js with a WebGLRenderer. An optional feature is to set markers on the image.

Dependencies

  • $document
  • $window
  • $timeout
  • SpatializeInterface
  • $debounce
  • $log

Directive Info

  • This directive creates new scope.
  • This directive executes at priority level 0.

Usage

  • as element: 
    <image-viewer
    src="string"
    [options="Object"]
    [spatialize]>
  ...
</image-viewer>

Arguments

Param Type Details
src String

Path/url to the image
options
(optional)
 Object

Force resolution/aspect ratio with width and height property
spatialize
(optional)
 Boolean

Activate spatialize feature, e.g. setting markers on image
