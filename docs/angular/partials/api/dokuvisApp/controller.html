Controller components in dokuvisApp

Name Description
configCtrl

Controller for configuration view.

Controller for main explorer view.

Controller for the navbar (visible at start page and project list page).

Controller for view to list and organize all projects.

Controller of the Project Home view, organizing subprojects and project/subproject information.

Controller for register view.
tasksCtrl

Controller for the project management view.
uploadCtrl

Controller of the Upload modal. The modal can be used to upload either a source with its metadata, or a 3D model.
