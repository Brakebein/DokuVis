sourcesList

  1. - directive in module dokuvis.sources

List all sources as a collection of thumbnail items. A blank item is provided as an entry point to upload further sources.

Dependencies

  • $state
  • ComponentsPath
  • SourcesCache
  • SourceUploader

Directive Info

  • This directive creates new scope.
  • This directive executes at priority level 0.

Usage

  • as element: 
    <sources-list
    [show-upload]>
  ...
</sources-list>

Arguments

Param Type Details
showUpload
(optional)
 Boolean

Enable the blank source item as an entry point to upload sources.

Example

<sources-list show-upload="can('historian')"></sources-list>
