Events

    Event that gets fired, when a task in the gantt chart has been selected. If task and row are not set, the event is supposed to trigger a deactivation of the current task.

    Type:

    broadcast

    Target:

    $rootScope

    row
    (optional)

    Row object of the task

  ...
