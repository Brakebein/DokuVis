dokuvisApp

Dokuvis base module.

Module Components

Object

Name Description
API

Constant defining the base url to API. It is injectable like any other service.
config

Configures ui.router states, state resolve functions, and some defaults.
run

Code execution after bootstrapping AngularJS.

Factory

Name Description
AuthenticationFactory

Service responsible for checking the user status.
From tutorial: http://code.tutsplus.com/tutorials/token-based-authentication-with-angularjs-nodejs--cms-22543
UserAuthFactory

Service responsible for contacting the login endpoint and validating the user (and logging out the user).
From tutorial: http://code.tutsplus.com/tutorials/token-based-authentication-with-angularjs-nodejs--cms-22543
TokenInterceptor

Service responsible for sending in the access token and the key along with each request to the server.
From tutorial: http://code.tutsplus.com/tutorials/token-based-authentication-with-angularjs-nodejs--cms-22543
Archive

$resource for archives.
Category

$resource for Category. A category can have multiple attributes (CategoryAttribute), which can be assigned to models.
API url: api/auth/project/:project/category/:id
CategoryAttribute

$resource for CategoryAttribute, that can be assigned to models. Usually in combination with Category.
API url: api/auth/project/:project/category/:cid/attribute/:aid
Comment

$resource for comments.
ConfirmService

Service providing a dialog with two buttons to confirm or abort an action.

// default values
alertDefaults = {
    backdrop: 'static',
    templateUrl: 'partials/alerts/confirmAlert.html',
    show: true
};
alertOptions = {
    abortButtonText: 'Abbrechen',
    actionButtonText: 'OK',
    headerText: 'Fortfahren?',
    bodyText: 'Sind Sie sicher?',
    type: 'warning'
};
Project

$resource for project related tasks.
ProjInfo

$resource for ProjInfo.
Source

$resource for sources/documents.
SpatializeInterface

Interface between imageViewer and webglView directives and spatializeModalCtrl to manage interaction across their different scopes.
Subproject

$http methods for subproject related tasks
Uploader

This factory creates an instance of FileUploader (angular-file-upload), so it can be used in multiple controllers. Authentication headers are set for uploading.
Utilities

This factory provides some helpful functions.
webglContext

This service initializes standard objects and materials for the 3D viewport and holds all mesh/geometry instances. So the 3D content is kept in the memory, even if the viewport (respectively webglView directive) gets destroyed, and doesn't need to be reloaded when the user returns to the 3D viewport. Materials etc. are also globally availlable via THREE.DokuVisTray.

Controller

Name Description
categoryEditModalCtrl

Modal controller for managing categories and their attributes.
configCtrl

Controller for configuration view.
explorerCtrl

Controller for main explorer view.
navCtrl

Controller for the navbar (visible at start page and project list page).
newArchiveModalCtrl

This controller handles the modal to enter a new archive to the system.
newProjInfoModalCtrl

Modal controller for creating or editing information assets (ProjInfo) of projects/subprojects.
newSubprojectModalCtrl

Modal controller for creating or editing subprojects.
projectlistCtrl

Controller for view to list and organize all projects.
projectModalCtrl

Modal controller for creating or editing projects.
projHomeCtrl

Controller of the Project Home view, organizing subprojects and project/subproject information.
registerCtrl

Controller for register view.
sourceDetailCtrl

Controller for the modal to display detailed information about sources.
uploadCtrl

Controller of the Upload modal. The modal can be used to upload either a source with its metadata, or a 3D model.

Directive

Name Description
horizontalScroll

Translates mousewheel event into horizontal scroll movement.
imageViewer

A viewer to inspect an image in detail. The user is able to zoom and pan. It builds on three.js with a WebGLRenderer. An optional feature is to set markers on the image.
rampSlider

A slider visualized as a box with a color ramp as background.
webglView

Directive implementing the 3D viewport and all 3D functionalities using http://threejs.org/.
