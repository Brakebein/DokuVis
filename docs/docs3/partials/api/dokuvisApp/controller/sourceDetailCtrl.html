sourceDetailCtrl

  1. - controller in module dokuvisApp

Controller for the modal to display detailed information about sources.

Methods

  • nextIten(incr);

    Load next (or previous, if negative) item. incr is relative to the current item. So 2 will load the next but one, -1 will load the previous one.

    Parameters

    Param Type Details
    incr Number

    number of steps to move (usually 1)

  • postComment();

    Save a new comment about the current item.

  • postAnswer(comment);

    Save a new comment of type answer.

    Parameters

    Param Type Details
    comment Object

    parent comment

  • enterGraph(id);

    Closes and destroys this modal and changes to GraphSearch state.

    Parameters

    Param Type Details
    id Number

    GraphSearch start Id

  • close();

    Closes the modal and destroys the controller instance/scope.
