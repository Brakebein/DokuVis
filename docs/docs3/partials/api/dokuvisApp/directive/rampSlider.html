rampSlider

  1. - directive in module dokuvisApp

A slider visualized as a box with a color ramp as background.

Dependencies

  • $document

Directive Info

  • This directive executes at priority level 0.

Usage

  • as element: 
    <ramp-slider
    ng-model="number"
    [ng-change="expression"]
    [rs-color="string"]
    [rs-color-end="string"]>
  ...
</ramp-slider>
  • as attribute: 
    <ANY
  ramp-slider
  ng-model="number"
  [ng-change="expression"]
  [rs-color="string"]
  [rs-color-end="string"]>
...
</ANY>

Arguments

Param Type Details
rampSlider Boolean
ngModel Number

Value the directive is bound to [0..1]
ng-change
(optional)
 expression

Expression to be evaluated after the value has changed
rs-color
(optional)
 String

Left (or base) color as hexadecimal color value
rs-color-end
(optional)
 String

Right color as hexadecimal color value

Example

<div ramp-slider
    ng-model="opacity"
    ng-change="setOpacity()"
    rs-color="#ddd"
    rs-color-end="#bbb" >
</div>
