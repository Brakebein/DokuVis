webglView

  1. - directive in module dokuvisApp

Directive implementing the 3D viewport and all 3D functionalities using http://threejs.org/.

Dependencies

  • $stateParams
  • $window
  • $timeout
  • webglContext
  • webglInterface
  • $rootScope
  • phpRequest
  • neo4jRequest
  • $http
  • $q
  • Utilities
  • Comment
  • ConfirmService
  • SpatializeInterface

Directive Info

  • This directive creates new scope.
  • This directive executes at priority level 0.

Usage

  • as attribute: 
    <ANY
  webgl-view="string"
  [nav-toolbar="Array"]
  [axis]
  [option-toolbar]
  [spatialize]>
...
</ANY>

Arguments

Param Type Details
webglView String

Id/name that is used for internal bindings (important, if there are multiple active webglView directives)
navToolbar
(optional)
 Array

Activate navigation toolbar on top of the viewport. Array elements define which features will be available, e.g. 'focus', 'move', 'shading', 'camera', and more to come.
axis
(optional)
 Boolean

Activate world axis in the left-bottom corner of the viewport
optionToolbar
(optional)
 Boolean

Activate extende options
spatialize
(optional)
 Boolean

Activate spatialize feature, e.g. setting markers on 3D model

Example

<div webgl="main"
     nav-toolbar="['focus', 'move', 'camera']"
     axis>
</div>
